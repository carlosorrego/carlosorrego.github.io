Trabajador ¶

API trabajador  ¶

Permite crear o modificar un trabajador presente en CTRL.IT utilizando como llave el campo “rut”.

Los campos que contiene el request son:

Nombre Descripcion
rut Rut del trabajador , sin puntos ni guion.
nombre Nombre del trabajador
apellido_paterno Apellido paterno del trabajador
apellido_materno Apellido materno del trabajador
especialidad Indica que tipo de trabajo o cargo tiene el trabajador
contrato Indica una agrupacion para las trabajadores dentro una empresa , puede ser asociado a una gerencia por ejemplo
area Indica otro tipo de agrupacion de trabajadores que puede estar asociado a las labores que desempeña
email Email del trabajador
rut_empleador Rut del empleador
razon_social Razon social del empleador
supervisor El jefe o supervisor del trabajador
tipo_costo Indica si es costo indirecto o directo
fecha_ingreso Fecha de inicio del trabajador en ese recinto con formato YYYY-MM-DD
fecha_termino Fecha de termino del trabajador en ese recinto con formato YYYY-MM-DD
fecha_turno Fecha a la cual se quiere definir el turno (este campo es opcional)
entrada_turno Hora de entrada del turno para la fecha definida en el campo fecha_turno
salida_turno Hora de salida del turno para la fecha definida en el campo fecha_turno
entrada_colacion Hora de entrada a colacion del turno para la fecha definida en el campo fecha_turno
salida_colacion Hora de salida a colacion del turno para la fecha definida en el campo fecha_turno
art22 Indica si el trabajador posee articulo 22 o no , si se requiere que tenga articulo 22 debe indicar TRUE en el valor del campo y en caso contrario solo el valor vacio.
recinto Corresponde a la clave del recinto a la que pertenece el trabajador, este es un valor unico interno de CTRL.IT

Vacaciones ¶

API vacaciones  ¶

Permite crear un registro de vacaciones para un trabajador utilizando como llave los campos rut, inicio y termino.

Los campos que contiene el request son:

Nombre Descripcion
rut Rut del trabajador , sin puntos ni guion.
recinto Corresponde a la clave del recinto a la que pertenece el trabajador, este es un valor unico interno de CTRL.IT
inicio Fecha de inicio de vacaciones del trabajador en ese recinto con formato YYYY-MM-DD
termino Fecha de termino de vacaciones del trabajador en ese recinto con formato YYYY-MM-DD
codigo Es un codigo indicatorio de las vacaciones como folio por ejemplo

API licencia  ¶

Permite crear un registro de licencias para un trabajador utilizando como llave los campos rut, inicio y termino.

Los campos que contiene el request son:

Nombre Descripcion
rut Rut del trabajador , sin puntos ni guion.

Permite consultar el resumen de asistencia de un recinto.
